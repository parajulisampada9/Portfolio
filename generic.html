Planning and Implementation

AWS Honeypot

This project will leverage AWS to demonstrate how to effectively secure cloud environments using industry-standard tools by a building honeypot system.

  • Set up a cloud-based honeypot using AWS services like EC2, VPC, and Security Groups to simulate a vulnerable environment and attract potential attackers.
  • Configured logging and automation by sending collected data to S3, triggering Lambda functions to parse logs and store key metadata in DynamoDB.
  • Utilized Python and Streamlit to process and visualize attack data, while incorporating tools like CloudTrail and AWS Health for monitoring and alerts.
  • Intentionally kept the system vulnerable and deployed decoy files to lure attackers, then monitored and analyzed activity over a week to identify attacker behavior and metadata.
